# **volunteer-tree**
*********************
## **概要**  
このアプリは、自分が参加したボランティア活動を記録し、共有することができるＷｅｂアプリケーションです。  自身の活動記録とあわせてボランティアとして参加したイベントの評価（業務内容の難易度ややりがいなど）を記録することができます。  ユーザー同士でイベントに関する評価を確認することができ、さらにチャットで情報交換や活動参加への誘いあいなども可能です。
********************************************************************************************
## **サイトテーマ**
「自己の活動記録を共有し、自己の歩みを讃えるとともに社会とのつながりを得る」
********************************************************************************************
#### **テーマを選んだ理由**
社会貢献活動は、特に若い人の中では、「意識が高い」「偽善である」と思われるのではないかという心配から、自身の活動について身近な友人に話しにくかったり、興味があっても参加する最初の敷居が高いのではないかと思います。　　ボランティア活動に特化した情報を集めることで、活動者の交流の場とするとともに、はじめてボランティア参加する方が気軽に情報を収集できる仕組みを提供します。
###### **【きっかけ】**
自分自身が、大きなイベントの主催を担当しており、イベント開催には多くのボランティアによる協力が不可欠であると実感するとともに、想像よりも多くの方がボランティア活動に興味を持っていると感じました。  ボランティアの方は、１回だけではなく様々なイベントに参加していることが多いため、そういった活動記録を残し、活動者同士で共有することで、より社会のつながりを高めることができるのではないかと考えたことがきっかけです。
###### **【問題点】**
* 自分の活動記録やボランティア参加証明書をまとめて記録しておくツールがない（就職活動などの際にまとめなおさなくてはならない)
* 活動記録が各自のSNSやブログに分散しており、検索しづらい
* ボランティア参加の難易度がわかりづらい
* 身近な友人を誘ってボランティアに参加するのは最初はハードルが高い
###### **【解決策】**
* ボランティア活動に特化した投稿システムを提供し、イベント名や活動内容、写真（活動証明書などの添付）を記録
* ユーザー同士で記録の共有や、チャット機能による交流ができる
* 参加したボランティアの難易度を★数で評価し、共有できる
*****************************************************************************************************
## **ターゲットユーザー**
* 社会貢献活動に興味があり、参加できるイベントを探している人
* ボランティアに何度も参加し、参加経験を簡単に記録したい人
* ボランティア仲間を探したい人
* 学生（就職活動や受験を控え自己アピールの材料を探している方）
* ボランティア側のニーズを知りたいイベント主催者
*****************************************************************************************************
## **主な利用シーン**
* はじめてボランティアに参加するとき
* 自分の活動記録を整理し、ほかのユーザと共有するとき
* 活動記録を自身の記録として提出するとき
* イベント主催者がボランティア側のニーズを把握し、ボランティアを募集したいとき
*****************************************************************************************************
## **利用方法**
* ユーザー新規登録を行う、またはログインする
* 活動記録とイベント評価を投稿
* ほかのユーザの投稿を閲覧し、気に入った投稿に「いいね」「コメント機能」を残す
* キーワード検索を使い、目的にあわせたイベントをチェック
* チャット機能を使い、ユーザー同士での会話を行う
******************************************************************************************************
